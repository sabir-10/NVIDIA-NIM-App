<h2>NVIDIA NIM RAG</h2>

A Retrieval-Augmented Generation (RAG) application built with Streamlit, LangChain, NVIDIA AI Endpoints, and FAISS that allows users to ask questions about a PDF document and receive accurate, context-aware answers.

<h3>Features</h3>

Load and process PDF documents

Split text into chunks using recursive text splitting

Generate embeddings using NVIDIAEmbeddings

Store and search vectors with FAISS

Answer questions using ChatNVIDIA (LLaMA instruct model)

Display retrieved document chunks for transparency

Interactive web interface built with Streamlit
